# streetwander

An agent takes creative walks through real neighborhoods via Mapillary street imagery. At each step it anchors briefly in the frame, then lets its thinking drift — into metaphors, parallel cases, small theories. The street is a stirring stick, not a subject. By the end, the mind has usually traveled further than the feet.

Every walk produces a main artifact and, sometimes, a few companion pieces — the shape of what got conjured in the wandering.

Each walk on this site is its own page: a map, the full per-step thinking, and the artifacts inline.

Live at <https://jontoews.com/streetwander/>.
