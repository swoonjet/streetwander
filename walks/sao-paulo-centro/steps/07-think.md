WANDER:
That strip is almost certainly addressable LED tape — probably sixty pixels per meter, each one individually controllable from a DMX board or an Art-Net packet — which means the viaduct is, technically, a very low-resolution screen that happens to be shaped like a bridge. A century of architectural lighting was about revealing a fixed object; this is a different category altogether. The bridge has a state variable now. Green today, red tomorrow, pulsing for the derby, dark for the outage, rainbow for a week in June. Sydney cracked the governance question first, almost by accident: there is, in fact, a formal Transport for NSW application to request a Harbour Bridge color, with criteria about which causes qualify, which don't, how far in advance, a whole civic hue protocol that somebody had to draft by hand after the LEDs were already up and humming. Paris has the equivalent for the Eiffel. Every major city is quietly building the legal scaffolding for the question of *whose cause rides public infrastructure this Tuesday night*, and mostly they're making it up as they go, which is how 1968 felt about traffic signs before the Vienna Convention, except that this time the installed base is fragmenting first and any standard will have to be retrofitted against a hundred different mayors' color-request PDFs. The deeper move is that buildings have become applications. They have runtime state, a backend, an administrator, a changelog — all the things a document doesn't have — and the profession that thought it was in charge of them has been quietly demoted to *frontend* while lighting controls became the backend nobody voted for. It migrates down-market faster than the discourse tracks: the parking garage of a suburban Ohio mall now has color-changing accents because the LED tape is cheaper than the static fixtures, the GC isn't thinking about politics, and then come October the mayor calls asking for pink. Suddenly facility management is fielding color tickets. Nobody decided this, which is the thing I keep circling — the commodity curve decided it, the way the commodity curve decided that everyone's toaster would have wifi before anyone had worked out what that meant.
